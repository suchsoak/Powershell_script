# Usage

>[!IMPORTANT]
>Don't forget, from the Get-ExecutionPolicy
>
>Unblock-File -Path .\script.ps1

**You can usage the script.ps1**

```sh
   .\script.ps1
```

>[!TIP]
>Or you can use the scripts separately
>

```sh
  .\wsl.ps1
  .\firewall.ps1
  .\browser.ps1
  .\drivereset.ps1
  .\netreset.ps1
  .\disk_verefication.ps1
  .\telssh.ps1
  .\language.ps1
  .\ide.ps1
  .\systeminfo.bat
  .\restorepoint.ps1
  .\fileinfo.ps1
  .\eventlog.ps1
  .\password.ps1
```
